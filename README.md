Un clavier matriciel 3x4 est un composant électronique très courant, 
utilisé pour permettre à un utilisateur de saisir des données numériques 
dans un système (comme un code d'accès ou une valeur de température). 
Il est souvent composé de 12 ou 16 touches organisées en 3 colonnes et 
4 lignes ou 4 colonnes et 4 lignes.
